Sobre este Projeto

Este projeto é uma demonstração educativa das 5 principais vulnerabilidades do OWASP Top 10, com exemplos interativos de ataques, explicações detalhadas sobre riscos e métodos de prevenção.

Objetivo

O objetivo é ajudar desenvolvedores e estudantes a entenderem os riscos de segurança em aplicações web e como implementar proteções adequadas.

Aviso Legal

As demonstrações neste projeto são simulações controladas. Não utilize as técnicas demonstradas em sistemas reais sem autorização.
